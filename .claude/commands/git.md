Build, commit, and push all changes to main. Steps:

1. Run `npm run build` to build frontend assets
2. Stage all changed files (source + build assets)
3. Write a concise commit message describing the changes
4. Commit and push to origin main
5. Tell the user what to run on the live server:
   ```
   cd /path/to/plutz-app
   git pull origin main
   ```
